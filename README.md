# Сервис для учета оплаты коммунальных платежей

- Front: Angular
- Back: PHP\Laravel

## Реализовано:
- CRUD квартир
- CRUD счетов
- CRUD шаблонов счетов
- CRUD денежных переводов
- Загрузка с внешнего API курсов валют, сохранение
